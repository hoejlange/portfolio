My Plants

My Plants gives the user reminders when the house plants need water. The user can scroll through the plant library and add the plants he/she has.
My Plants is a single page web application build in HTML, CSS and JavaScript. The web app uses Firebase for login authentification and data storage, which allows the user to save plants to the profile.

Keywords: Data Storage, Single Page Application, Web Application, User Journey

Klimaspillet

Klimaspillet is a web app made for children in sixth grade visiting the greenhouses in the Botanical Gardens in Aarhus. Klimaspillet is a quiz that follows a specific route through the greenhouses. The purpose of the web app is an interactive learning platform for the users to learn about sustainability.
The solution is coded with HTML, CSS and JavaScript.

Keywords: Interactivity, UX, Usability, Web Application
